Dock and window icon for the 3DRecon GUI

1. Add a PNG image named:  icon.png
   Recommended size: 256×256 or 512×512 pixels.

2. The GUI will use it for:
   - The window title bar and Cmd+Tab
   - The Dock icon on macOS (if pyobjc is installed: pip install pyobjc-framework-Cocoa)

If icon.png is missing, the default Python/app icon is used.
